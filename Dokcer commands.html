Containers and Images Commands

docker images - list all images docker rmi - delete an image docker rm - delete a container docker image prune - Remove unused images docker pull - Pull an image from DockerHub docker run - Create and run a new container docker run -it - run and input int the container docker run -d -e MYSQL_ROOT_PASSWORD=secret - run mysql container in detached mode with environment variable docker run -d -e MYSQL_ROOT_PASSWORD=secret --name mysql docker ps -a - list all Local container docker ps - list all running containers

Port binding Commands

docker run -p: - This line is a Docker command used to run a container and map ports between your host machine and the container. (8080:3306)

Troubleshoot Commands

docker log - This command displays the logs (output) from mention container. docker exec -it bin/bash -This command is used to open an interactive shell session inside a running Docker container. docker exec -it bin/sh - if there is no bash use it their (search env,ls,etc)
